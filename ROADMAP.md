# THESEUS ‚Äî Roadmap
*The master plan. Living doc ‚Äî updated as we go (newest status on top). Tactical board: `KANBAN.md`. Lanes: `docs/TEAM_LANES.md`.*

**Mission:** the **accreditable fleet-learning layer for unmanned maritime vehicles (UUVs) under DDIL** ‚Äî each vehicle learns onboard (human-in-command), and a fleet node coordinates the improvements as **model deltas (never raw data)** via a provenance-gated, eval-gated, signed merge. Real on real, **every model update + human decision sealed as DoD-standard attestation evidence** (in-toto / SLSA / Sigstore + NIST **OSCAL** via Lula). *Surface-contact Pattern-of-Life (NV063) is the adjacent funded track; the same spine runs on big surface combatants (DDG/CG) too.*

**Architecture ‚Äî Jun 17 pivot (`docs/INTEGRATION_SPEC.md`):** compose best-of-breed open components that pass the **GATE** (permissive ¬∑ airgap-capable ¬∑ edge-feasible) ‚Äî Stone Soup (fusion), Eclipse Zenoh + DDS plugin (DDIL transport *and* the SWAN-side ship-bus bridge), PyOD/Merlion/River (cold-start detection), MLflow + ONNX (lifecycle), UDS/Zarf/Pepr/Syft/cosign (deploy, already verified) ‚Äî and **build only the ~15% non-commodity inch**: onboard cross-system fusion under DDIL ¬∑ the **runtime-decision attestation extension** ¬∑ the DDIL delivery loop ¬∑ Navy domain knowledge. Adopting the standards *is* the moat-sharpener ‚Äî a recognized, interoperable record is what makes other vendors emit *into* it. (Internal Merkle hash-chain stays as the implementation detail; the external artifact is standard attestations.)

## ‚òÖ North Star ‚Äî the fleet-learning flywheel (the 10/10 vision ¬∑ `docs/vision/FLEET_LEARNING_VISION.md`)
The founder's vision, and the apex this whole build climbs toward: **each ship is a self-contained city.** Every subsystem (machinery / power / propulsion / nav / damage-control / contacts) runs its own onboard ML + anomaly detection and **recommends to the watch ‚Äî human always in command**, fully airgapped under DDIL. When the ship is **not in a contested environment**, it syncs **model deltas (never raw sensor data)** to a **fleet brain** (Blackwell / shore); the fleet improves as one ‚Äî **Tesla-fleet-style, but human-authorized, eval-gated, and provenance-attested.**
- **The unification (why this scores 10):** the tamper-evident record is not a side feature ‚Äî it is *what makes fleet learning accreditable*. It kills federated learning's two killers: **model-poisoning** (the fleet brain merges only provenance-attested deltas ‚Äî reject a captured/compromised node's contribution) and **"you can't accredit a model that changes"** (you accredit the *pipeline's provenance*, not the frozen weights = **cATO-for-AI**, in the AO's own OSCAL language). The moat we already built and the biggest version of the mission are the **same artifact.**
- **We're not fighting the Navy's roadmap ‚Äî we close it.** DECK / Project Overmatch *open* the loop (collect at the edge ‚Üí ship ashore ‚Üí retrain ashore); **THESEUS closes it** ‚Äî learn *onboard* under DDIL + the safe, provenance-gated fleet-merge. THESEUS is the onboard payload the Warfighting Data Ecosystem (and UDS Fleet) surfaces.
- **Open lane ‚Äî VERIFIED (Jun 17, sourced; `docs/research/DECK_BLUE_OCEAN.md`):** no named Navy AI-model **provenance/accreditation layer** exists today ‚Äî DECK is a data pipeline (no model lineage), cATO is systems-level not model-level, and CDAO's AI-assessment framework isn't due until **~Jun 2027** ‚Üí an **~18-month governance gap** where fleet learning ships with no accreditable record. *That gap is THESEUS's lane.* **Honest:** DECK's *internal* provenance isn't public and the integration isn't proven ‚Äî claim the gap, not the integration. *(Acquisition/engagement is the founder's lane ‚Äî handled.)*
- **Deliver now, demonstrate the vision:** ship the one-ship demo for the event, and **demonstrate the flywheel in miniature** ‚Äî 2 simulated ships + a fleet brain (Mac = Node 3; Blackwell optional, *not needed* for the prototype), each learns locally, "safe to sync," signed deltas, the fleet brain **merges only the attested ones (reject a poisoned delta live)**, eval-gate, push back, both improve, every step sealed. The whole vision on a table, disconnected. *(The fleet brain + real DECK integration is the longterm arc ‚Äî Phase 3+.)*
- **Language discipline (carry into the room):** "*recommended* intervention, human-in-command" (never "the fleet updates itself"); "improves *safely* ‚Äî every update eval-gated + attested + rollback-protected" (catastrophic forgetting is the real risk; those three are the answer).
