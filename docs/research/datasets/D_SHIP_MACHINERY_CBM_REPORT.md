# D — Ship / Marine Machinery + Condition-Based Maintenance (CBM) Datasets

**Category:** ship/marine machinery, condition-based maintenance, predictive maintenance, machinery-anomaly telemetry (THESEUS SWAN-side HM&E feed).
**Discipline:** OPEN-SOURCE / OPEN-ACCESS only. LICENSE-FIRST. Every dataset + URL verified via web fetch or web search; anything not directly confirmed is labeled **UNVERIFIED**.
**Bottom line up front:** Real, in-situ, naval shipboard HM&E telemetry is essentially **not open** (navy / shipowner proprietary). The best open path is (1) the **simulator-generated frigate GT propulsion** set [UCI CBM], (2) **real operational compressor/pneumatic** telemetry from an adjacent domain [MetroPT-3], and (3) the **gas-turbine degradation/RUL** benchmark [NASA C-MAPSS] — then transfer to ship GT/aux machinery.

## Access mechanics (top datasets)

**1. CBM of Naval Propulsion Plants (UCI #316) — THE BULLSEYE.**
Direct anonymous download from UCI (553.8 KB zip: `data.txt` + `README.txt` + `Features.txt`), or programmatically via `from ucimlrepo import fetch_ucirepo; fetch_ucirepo(id=316)`. No registration. License is explicitly **CC BY 4.0** on the UCI page ("sharing and adaptation for any purpose, provided appropriate credit") — clean for **both product and SBIR commercial use**. Cite Coraddu, Oneto, Ghio, Savio, Anguita, Figari (2014). 16 sensor features (torque, rpm, temps, pressures, fuel flow) + two ground-truth decay coefficients (compressor kMc, turbine kMt) over a 3–27 kn speed grid. The single most on-target open set for THESEUS SWAN-side propulsion decay.

**2. MetroPT-3 (UCI #791).** Anonymous download (208 MB zip: one CSV + a data-description PDF) from UCI, mirrored on Zenodo (records 6854240 / 7766691 for MetroPT/MetroPT2). No registration. **CC BY 4.0 → commercial-clean.** 1.5 M rows at ~1 Hz from a compressor Air Production Unit; the stream itself is unlabeled but four real air-leak failure windows (Apr–Jul 2020) with maintenance timestamps are published in the dataset table, so you can build supervised or anomaly-detection evaluations. Sensor set (pressures, motor current, oil temp, valve electrical signals) is essentially the shipboard air-compressor / pneumatic / HVAC sensor class.

**3. NASA C-MAPSS.** The run-to-failure degradation datasets (FD001–FD004, each train/test/RUL `.txt`) are donated to the NASA Prognostics Center of Excellence (PCoE) repository and are US-Government work (no copyright; free and open with citation). The official NASA page is intermittently offline; the **PHM Society mirror** (`data.phmsociety.org/nasa/`) and Kaggle copies are reliable. Note the distinction: the **C-MAPSS *simulator software*** is download-restricted by NASA Glenn, but the **degradation *datasets*** are fully open. 21 sensor channels make it the canonical RUL/degradation benchmark; gas-turbine degradation physics transfer to GT-propelled warships.

**4. CWRU Bearing.** Per-file `.mat` downloads from the Case Western Bearing Data Center (12k/48k samples-sec drive-end + fan-end, seeded EDM faults). **No open license is posted** — it is universally used in academia but commercial embedding is legally ambiguous; treat as internal R&D/benchmark only until written permission is obtained. Zenodo mirrors (e.g., record 10987113) re-host but do not add a license.

**5. MIMII.** Zenodo download (100.2 GB original, split by machine type and SNR; DUE/DG editions are smaller). **CC BY-SA 4.0** — commercial use is permitted but **ShareAlike** means any derivative *dataset/adaptation* must be released under the same copyleft terms, which is a real trap for a closed product; route through legal before embedding. Acoustic (not telemetry) modality, but pumps/valves/fans map cleanly to shipboard auxiliary machinery for an acoustic-anomaly sensing channel.

## TOP PICKS for THESEUS

1. **CBM of Naval Propulsion Plants (UCI #316) — #1, the closest open thing to real shipboard HM&E.** It is literally a frigate gas-turbine (CODLAG) propulsion plant with the exact SWAN-side signal classes THESEUS watches (shaft torque, rpm, turbine temps/pressures, fuel flow, propeller torque) plus ground-truth compressor/turbine decay labels. **CC BY 4.0 = clean for product + SBIR.** Only caveat: it is simulator-generated and steady-state, so pair it with real-sensor sets for transient/noise realism.

2. **MetroPT-3 (UCI #791) — #2, best REAL-sensor proxy.** This is genuine operational compressor/pneumatic telemetry (not simulation) with real failure events and a maintenance log — the strongest open stand-in for shipboard air/HVAC/pneumatic + damage-control air systems, and **CC BY 4.0 commercial-clean.** Use it to prove the "detect impending failure before the watch" beat on real, noisy, drifting sensor streams.

3. **NASA C-MAPSS — #3, the prognostics-engine benchmark.** Open, citation-only, gas-turbine RUL gold standard; ideal for validating THESEUS's degradation/remaining-useful-life modeling before transferring to ship GT. Add **NASA IMS** + **FEMTO/PRONOSTIA** for true run-to-failure *bearing* RUL (pumps/motors/shafts), with the license caveat below.

**Rotating-aux layer (pumps, motors, generators, fans):** CWRU + MAFAULDA + FEMTO cover the universal bearing/imbalance/misalignment fault classes that recur across shipboard rotating machinery — but all three are **license-unclear or copyleft**, so use them for internal R&D and benchmarking, not product-embedded data, until cleared.

## Access / gotchas note

- **License tiers for commercial (product + SBIR):**
  - **Clean ✅:** UCI CBM Naval (CC BY 4.0), MetroPT-3 (CC BY 4.0), NASA C-MAPSS / IMS (US-Gov, citation only).
  - **⚠️ Copyleft trap:** MIMII (CC BY-SA 4.0) — derivative datasets must be shared-alike; legal review before embedding.
  - **🚩 License unstated / research-only / verify:** CWRU Bearing, MAFAULDA, FEMTO/PRONOSTIA, PHM Society challenges, Strathclyde hull/prop/GT — fine for internal benchmarking; **do not ship in a commercial product without written permission.**
- **NASA PCoE page is intermittently down** → use the PHM Society mirror (`data.phmsociety.org/nasa/`).
- **C-MAPSS nuance:** datasets open; the simulator *software* is restricted. Don't conflate them.
- **The honest gap:** real, in-situ **naval/commercial shipboard HM&E telemetry is not openly published** (navy + shipowner proprietary; most marine-engine anomaly papers use private vessel data they do not release). The viable open strategy is **simulator-naval (UCI CBM) + real-adjacent-domain (MetroPT-3 compressor, C-MAPSS GT) + generic rotating-machinery (CWRU/FEMTO/IMS)**, then domain-transfer. Flag this gap to the SBIR/product team early.
- **UNVERIFIED items** (flagged in-table): Strathclyde hull/prop/GT license; exact MIMII-DUE vs MIMII-original license split; per-challenge PHM Society commercial terms. Confirm before relying on these for commercial scope.
